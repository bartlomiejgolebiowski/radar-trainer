# Radar Trainer — WSA Duisburg

Lern-App für die Radarprüfung (WSA Duisburg): 83 Multiple-Choice-Fragen, Training mit intelligenter Fehlerwiederholung und Probeprüfung mit 60-%-Bestehensgrenze. Fortschritt wird lokal im Browser gespeichert.
